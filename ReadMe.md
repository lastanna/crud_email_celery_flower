Модель: Создайте Django-модель Lesson (Урок) c полями на ваше усмотрение.

Бизнес-логика: Реализуйте механизм, который отслеживает важное событие в жизненном цикле урока — например, его создание или завершение.

Фоновая задача (Celery): Когда это событие происходит, должна асинхронно запускаться задача Celery, что будет имитировать отправку уведомления студенту (например, просто логировать сообщение: "Уведомление отправлено студенту [ID студента] по уроку [Название урока]").

Что предоставить: Ссылку на Git-репозиторий (GitHub/GitLab) с вашим решением. Краткий README.md с инструкциями по запуску (включая Celery) обязателен.

Порядок выполнения задания
1) Создадим cbv для создания и редактирования задач.
2) Создадим сигналы, которые будут отправлять пользователю уведомления и emailы при возникновении важных событий:
создание событий и завершение.
3) Cоздадим docker и docker compose для запуска для запуска django app, redis и celery
При создании или завершении урока django post_save signal создает оповещение, которое передается в реальном времени
 по вебсокету (django channels). В это случае пользователю не нужно ждать, когда он залогинится в обучающую платформу
 для получения уведомления. Запуск вебсокета и передача оповещения осуществляется асинхронно. Для этого создается фоновая
 задача Celery, что позволяет выполнить запросы по сети паралельно за счет переключения контекста c исполнением кода и 
 не тратить время основного потока на выполнение этой операциию У вебсокета есть
первоначальный HTTP-запрос для "рукопожатия" (handshake), после которого устанавливается постоянное двустороннее соединение, 
 позволяющее обмениваться сообщениями (пакетами данных) в реальном времени без закрытия соединения

В описании задачи немного не понятно, кто создает уроки и кто их завершает.
Буду считать, что ситуация такая: администратор может создавать уроки, при создании
урока всем ученикам рассылается оповещения о создании урока. Каждый ученик может завершить урок, когда его пройдет, при
этом ему приходит уведомление о завершении урока.
